Available endpoints

Users

  • /auth/token/
  • /users/me/ - get information about me
  • /users/all/ - get list of all users
  • /users/add/ - add user

Tasks

  • /tasks/create/ - create new task
  • /tasks/all - get list of all tasks of all users
  • /tasks/user_id/{user_id}/ - get list of all tasks of user with user_id
  • /tasks/me/ - get all my tasks

Groups

  • /groups/add/ - create new group
  • /groups/all/ - get list of all groups
  • /groups/user_id/{user_id}/ - get all groups of user with user_id
  • /groups/me/ - get all my groups
